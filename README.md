# starter

## 简介

`starter` 是一个围绕**点睛网**相关内容进行整理与沉淀的资料型仓库，目标是帮助学习者、从业者以及对相关课程与工具感兴趣的用户，更高效地获取学习入口、梳理知识脉络，并沉淀一套可复用的资料管理方式。📚

仓库聚焦于两类内容：

- **学习资料整理**：包括常见学习主题、课程线索、资料分类建议、使用心得与学习路径参考；
- **实用工具汇总**：包括辅助检索、笔记整理、学习管理、资料归档等能够提升效率的工具与方法。

相比单纯地“堆链接”，本项目更强调**结构化、可持续维护、便于查找**。无论你是想快速入门，还是希望建立长期学习资料库，这个仓库都可以作为一个轻量但实用的起点。

## 使用说明

为了让仓库更易维护，建议按照统一规则使用与更新内容。✍️

### 资料整理建议

1. **按主题分类**  
   例如：基础入门、进阶学习、实务应用、案例解析、效率工具等。

2. **保持条目简洁**  
   每条资源建议包含：
   - 资源名称
   - 简短说明
   - 适用人群
   - 推荐理由
   - 更新时间（可选）

3. **优先保留高质量链接**  
   对重复、失效、信息密度低的内容进行筛选，避免仓库变成简单的链接堆积区。

4. **增加个人注释**  
   例如：
   - 适合入门还是进阶
   - 学习时长预估
   - 是否需要前置知识
   - 是否适合做资料收藏

### 工具收录建议

对于工具类内容，可采用以下格式：

```md
- 工具名称：
- 适用场景：
- 核心功能：
- 使用门槛：
- 推荐指数：
```

这样能够帮助后来者快速判断是否值得使用。

### 协作建议

如果你希望参与维护，建议遵循以下原则：

- 新增内容前先查看是否已有同类条目；
- 链接说明尽量准确，避免只有标题没有上下文；
- 提交前简单检查 Markdown 格式与链接可用性；
- 对争议较大的资源，可以先开 Issue 讨论。🤝
